# 🐍 Python Bootcamp Tutorial

A comprehensive, hands-on Python learning path — from absolute fundamentals to building real-world web applications with Flask, and finishing with an AI-powered capstone project.

This repository is organized as a structured bootcamp, with each numbered folder building on the concepts from the last. Whether you're a complete beginner or brushing up on core skills, you can work through the modules in order or jump directly to the topic you need.

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Yash-kitcoek/Python-Bootcamp-Tutorial.git
   cd Python-Bootcamp-Tutorial
   ```

2. **Set up a virtual environment** (recommended)
   ```bash
   python -m venv venv
   source venv/bin/activate   # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies** (for Flask modules)
   ```bash
   pip install flask
   ```

4. **Navigate to a module and run the code**
   ```bash
   cd 11_flask_basics
   python app.py
   ```

## 📖 How to Use This Repo

- Beginners: start at `01_Python Fundamentals` and work sequentially through to `10_Hands_on_projects`.
